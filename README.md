Project Title

IIUC Campus Network System

Description

This project replicates the International Islamic University Chittagong (IIUC) campus network using Cisco Packet Tracer.
The network design separates Male Campus and Female Campus, with Core, Distribution, and Access layers implemented for proper hierarchy and traffic management.

Objective

Understand campus network design and structure.

Practice VLAN setup for Male and Female campuses.

Configure inter-VLAN routing and verify device connectivity.

Simulate a campus network using Cisco Packet Tracer.

Tools & Technologies

Cisco Packet Tracer (version 8.x recommended)

Network devices: Routers, Switches, PCs, Printers

Protocols: VLAN, Inter-VLAN Routing, DHCP, Static IP
